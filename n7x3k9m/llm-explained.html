The Big Picture
An LLM does ONE thing: predict the next word. That's it. Everything else emerges from that.
Input

All the text so far (your prompt + what it already generated)

Output

ONE word (token). Then it feeds that word back and predicts the next one.

💡 Mind-blowing fact: ChatGPT doesn't "understand" your question, plan an answer, then write it. It literally guesses one word at a time, and coherent answers emerge from the pattern.

Step 1: Tokenization 🔪
Split your text into small pieces the model can understand
You type:
"What are cats?"
Tokenizer splits it into pieces:
What→2061
are→527
cats→24211
?→30

🤔 Why not full words?

Long/rare words get split: "unbelievable" → "un" + "believ" + "able". This way the model only needs ~50,000 tokens instead of millions of words.

💡 Not a neural network!

The tokenizer is just a lookup dictionary. No AI here — it's a simple find-and-replace. The AI starts at the next step.

Step 2: Embeddings 📍
Convert each token number into a vector that captures its meaning
Each number becomes a list of 4,096 numbers:
2061 ("What") → [0.12, -0.84, 0.31, 0.67, ..., -0.22]
527 ("are")   → [0.45, 0.10, -0.63, 0.18, ..., 0.91]
24211 ("cats") → [-0.33, 0.92, 0.21, -0.55, ..., 0.14]
30 ("?")     → [0.01, -0.11, 0.05, 0.03, ..., -0.08]
Similar words land close together in this space:
cat 🐱
dog 🐕
hamster
king 👑
queen 👸
prince
pizza 🍕
burger 🍔
sushi 🍣
run 🏃
walk 🚶
4096D space (shown as 2D)

✨ king − man + woman ≈ queen — The meaning is encoded in the math! Vectors learn this from reading billions of sentences.

Step 3: Self-Attention 👀
Each word looks at every other word and asks: "How relevant are you to ME?"
The bank by the river was flooded
bank → river: 0.65 bank → flooded: 0.25 bank → The: 0.05

Result: "bank" now knows it means riverbank, not a money bank. The attention scores told it!

How? Q, K, V

Each word makes 3 vectors:
Query = "What am I looking for?"
Key = "What do I contain?"
Value = "Here's my actual info"
Score = Query · Key (dot product)

Step 4: Feed-Forward Neural Network 🧮
After attention, each position goes through a network of neurons that "thinks deeper"
What each neuron does:
output = ReLU(W × input + b)

W = weight (how strong the connection is)
b = bias (the neuron's preference)
ReLU = if result < 0, output 0. Else, pass it through.

Scale:

Each layer has ~16,000 neurons
Each neuron connects to 4,096 inputs
That's 65 million weights per layer
× 96 layers = 6.2 billion just for feed-forward!

💡 This is the "knowledge storage." While attention figures out relationships between words, the feed-forward network stores facts like "Paris is the capital of France" in its weights.

Why 96 layers?

Early layers learn simple patterns (grammar, word types). Middle layers learn facts and relationships. Deep layers learn reasoning and complex logic. More layers = more "depth of thought."

Layer Examples:

Layer 1-10: "cats" is a noun, plural
Layer 30-50: "cats" are animals, pets
Layer 70-96: user is asking a question → generate informative answer

🔑 This stack of 96 blocks IS the neural network. The tokenizer and embedding lookup are just input wiring. Everything interesting happens in these blocks.

Step 5: Output — Predicting the Next Word 🎯
The final layer converts the hidden state into a probability for every word in the vocabulary

The Autoregressive Loop 🔄
It writes ONE word at a time, feeds it back, and repeats
It NEVER looks ahead

It can only see words to the LEFT. Like writing with a blindfold — each word is a fresh guess based only on what came before.

When does it stop?

It outputs a special [STOP] token. Or it hits the max length limit (e.g., 4096 tokens). That's why sometimes responses get cut off!

Temperature 🌡️
Controls how "creative" vs "predictable" the model is when picking the next word
🧊
T = 0
Always pick the top word
are 72% ✓
were 14%
have 8%
Predictable, deterministic.
Best for code & facts.
♨️
T = 0.7
Mostly top words, some variety
are 52% ← usually
were 28% ← sometimes
have 15%
Creative but coherent.
ChatGPT default ≈ 0.7
🔥
T = 2.0
Nearly random from all words
are 30%
were 27%
have 25% ← 🎲 picked!
Chaotic, often nonsense.
Fun for brainstorming.

💡 Temperature adjusts the probability distribution. T=0 makes the highest bar take everything. T=2 flattens all bars to nearly equal. The math: divide all scores by T before softmax.

How Was It Trained? 🎓
3 stages, from baby to expert — each stage builds on the previous
👶
Phase 1: Pre-training (Read the internet)
Feed it trillions of words from books, Wikipedia, websites, code. For each word, it guesses the next one. If wrong → adjust all 1.8T weights slightly. Repeat billions of times.
~$100M+ cost | 3-6 months | 10,000 GPUs
🧒
Phase 2: Instruction Fine-tuning (Learn to be helpful)
Humans write 100,000+ example Q&A pairs: "Q: What is gravity? A: Gravity is a force that..." The model learns the format of being a helpful assistant instead of just predicting text.
~$1M cost | Days-weeks | 100s of GPUs
🧑‍🎓
Phase 3: RLHF (Learn right from wrong)
Model generates multiple answers. Humans rank them (best → worst). A "reward model" learns what humans prefer. Then the main model is optimized to get higher rewards. This is why ChatGPT refuses harmful requests.
~$500K | Weeks | Ongoing process

🧠 What IS training technically? Adjusting 1.8 trillion numbers (weights) so the model's predictions match the correct answers. The algorithm: backpropagation — compute the error, then send it backwards through all layers, nudging each weight to reduce the error.

The Complete Picture 🗺️
Everything together — from your question to the answer
1
Tokenize: Split your text into pieces → numbers. Just a dictionary lookup, no AI.
2
Embed: Convert each number into a 4,096-dimension vector that captures meaning. Similar words = close vectors.
3
Self-Attention: Every word looks at every other word. "bank" sees "river" and knows it's a riverbank. Q·K scores determine relevance.
4
Feed-Forward: Each position goes through a network of neurons. This is where "facts" are stored in the weights. output = ReLU(W×input+b).
×96
Steps 3+4 repeat 96 times. Early layers: grammar. Middle layers: facts. Deep layers: reasoning. This stack IS the neural network.
5
Output: Score all 50,000 possible next words. Pick one (based on temperature). Feed it back in. Repeat until [STOP].

🎓 Training = finding the best 1.8 trillion W and b values, using backpropagation across trillions of words.
🔮 Inference = using those frozen values to predict words, one at a time, in the autoregressive loop.
